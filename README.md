# JSX Runtime Experiment

JSX(JavaScript XML) 런타임의 내부 동작을 학습하고 실험하기 위한 프로젝트입니다.

## 🚀 시작하기

프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따르세요.

1.  **의존성 설치**

    ```bash
    npm install
    ```

2.  **빌드 (TypeScript 컴파일)**

    `package.json`에 정의된 빌드 스크립트를 실행합니다. (일반적인 예시)

    ```bash
    npm run build
    ```

3.  **실행**

    빌드된 JavaScript 파일을 실행합니다.

    ```bash
    node dist/index.js
    ```
